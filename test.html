Сравнение технологий компоновки

CSS Grid Layout

Двумерная система компоновки, которая позволяет работать с колонками и строками одновременно. Идеально подходит для создания сложных макетов страниц.

CSS Flexbox Layout

Одномерная система компоновки, которая позволяет распределять элементы вдоль одной оси (горизонтально или вертикально). Отлично подходит для компонентов интерфейса.
